## 2. Calculating PV FV

### Single Sum

| Calculation   | Function          | FV/PV Factor  |
| ------------- | ----------------- | ------------- |
| Future Value  | `FV=PV(1+I/Y)^N`  | `(1+I/Y)^N`   |
| Present Value | `PV=FV/(1+I/Y)^N` | `1/(1+I/Y)^N` |

Hence, PV< FV when discount rate>0.
